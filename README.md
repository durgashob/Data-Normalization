# Database Normalization Project

This project demonstrates the normalization process of a student club membership database.

## Tasks Included

* Identifying data problems in the given table
* Converting the table to First Normal Form (1NF), Second Normal Form (2NF), and Third Normal Form (3NF)
* Designing an ER Diagram
* Writing SQL queries
* Performing JOIN operations

## Purpose

The purpose of this project is to understand how database normalization reduces redundancy, improves data accuracy, and organizes data efficiently.

## Features 
- Dockerized MySQL setup for portability and reproducibility
- Stepwise normalization of student club membership data (1NF → 3NF)
- ER diagram representation of normalized tables
- SQL queries including JOIN operations for relational data retrieval
- Practical demonstration of solving redundancy, update, insertion, and deletion anomalies
